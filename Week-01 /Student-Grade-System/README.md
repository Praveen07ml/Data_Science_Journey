# Student Grade Management System

## Overview

This is a simple Python program that manages student grades.
The program allows users to add students, view their grades, and analyze class performance.

This project was built as a practice exercise to understand how real programs are structured using Python basics.

---

## What the Program Can Do

The program provides the following features:

* Add a new student with marks
* Show all students and their grades
* Calculate the class average
* Find the top student in the class
* Display students who scored above the class average
* Exit the program through a menu system

The program runs in a menu-driven interface where the user selects options from the terminal.

---

## Concepts Used

This project focuses on strengthening core Python fundamentals:

* Dictionaries for storing student data
* Functions for organizing program logic
* Loops for running the menu continuously
* Conditional statements for decision making
* Basic input handling from the user

The goal was to connect these concepts to build a complete working program.

---

Learning programming is not only about syntax.
It is also about understanding how to break a problem into smaller parts and organize the solution.

This project was created to practice:

* structuring a program
* separating logic into functions
* designing a simple interactive application
